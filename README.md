# 计算机视觉实验三：图像下采样缩小、恢复与频域分析

## 实验环境
- 操作系统：Ubuntu / WSL2
- 开发工具：VS Code
- 编程语言：Python 3.12
- 依赖库：VS Code + Ubuntu (WSL) + OpenCV+NumPy+Matplotlib+SciPy

## 实验目的
1.实现图像直接下采样与高斯平滑后下采样，理解混叠失真与抗混叠处理的意义。
2.使用最近邻、双线性、双三次三种插值方法对缩小图像进行上采样恢复，对比不同插值效果。
3.用MSE、PSNR在空间域对恢复图像做客观质量评价。
4.通过FFT 频谱和DCT 变换从频域分析下采样与恢复过程中高频信息的丢失与重建情况。

## 实验原理与实现步骤
1. 图像下采样
下采样是将图像分辨率降低的操作，本实验实现了两种方式：
（1）直接下采样：直接对原始图像进行 1/2 缩放，使用最近邻插值，会保留原始高频细节，但易产生锯齿、混叠失真。
（2）高斯平滑后下采样：先对图像进行 5×5 高斯滤波（σ=1.5），滤除高频噪声和细节，再进行下采样，可有效抑制混叠，图像更平滑。
2. 图像恢复（上采样）
将下采样后的图像恢复到原始尺寸，对比三种经典内插方法：
（1）最近邻内插：直接复制最近邻像素值，计算最快，但会产生明显锯齿、块效应，边缘不连续。
（2）双线性内插：基于 2×2 邻域像素做线性插值，平滑效果优于最近邻，计算量适中，边缘更自然。
（3）双三次内插：基于 4×4 邻域像素做三次多项式插值，保留细节能力最强，边缘最平滑，视觉效果最好，但计算量最大。
3. 空间域质量评价
使用均方误差（MSE）和峰值信噪比（PSNR量化恢复效果：
MSE：衡量恢复图像与原始图像的像素差异，值越小，恢复效果越好。
PSNR：基于 MSE 的客观评价指标，单位为 dB，值越大，图像质量越好。
4. 傅里叶变换（FFT）分析
通过二维傅里叶变换将图像从空间域转换到频域，中心化后得到幅度谱：
（1）频谱中心对应低频分量（图像的整体亮度、平滑区域），边缘对应高频分量（图像的边缘、细节、噪声）。
（2）对比原始图、缩小图、恢复图的频谱，分析高频成分的丢失与恢复情况。
5. DCT 变换分析
离散余弦变换是图像压缩的核心变换，将图像转换到频域后：
（1）低频系数集中在左上角（本实验做了中心化处理，集中在中心），包含图像的主要能量；高频系数对应细节、噪声。
（2）计算低频区域（前 10% 系数）的能量占比，量化不同恢复方法对图像能量分布的影响。

## 实验结果
1.下采样、恢复图像自动保存到images/目录，空间域对比图、频谱图、DCT 系数图自动保存到results/目录，终端会打印 MSE/PSNR 指标、低频能量占比等关键结果。
2.空间域双三次插值恢复效果最好，MSE 最小、PSNR 最高；双线性次之；最近邻插值块效应与锯齿最明显，指标最差。
频域（FFT）下采样会丢失大量高频分量；最近邻恢复频谱伪影重，双三次频谱更接近原始图像。
DCT自然图像能量主要集中在低频；双三次恢复的低频能量占比与原始图像最接近，质量最优。
先高斯平滑再下采样能有效减少混叠，提升后续恢复质量。
